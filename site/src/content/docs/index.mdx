## What is mixr

mixr is a terminal-native DJ app for electronic music. It pulls tracks from Beatport, runs two decks with a real crossfader + EQ + filter chain, accepts input from popular USB DJ controllers, and uses an AI assistant to suggest mixes from your library based on key and BPM.

## The family

mixr is one of three small Rust tools by Chris McLennan:

- **mixr** — this DJ app
- **[tmnl](https://tmnl.sh)** — a GPU-rendered terminal that lets mixr render as a native tab (with full graphics) instead of nested in some other terminal emulator
- **[mnml](https://mnml.sh)** — a NvChad-style terminal IDE

mixr runs fine in any terminal. Inside tmnl specifically, mixr gets native-tab integration via the `tmnl-protocol` cell-streaming layer — meaning the crossfader and EQ knobs render as proper graphics, not unicode block art.
