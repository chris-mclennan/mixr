## Script the whole rig

mixr exposes a file-based IPC at `~/.mixr/command`. Drop a JSON line, mixr reads + acts. Works from any language, any tool:

```sh
echo '{"skip":1}'           > ~/.mixr/command   # next track
echo '{"crossfader":-0.5}'  > ~/.mixr/command   # fader toward deck A
echo '{"transition":"echoout"}' > ~/.mixr/command   # force the next mix
```

Used internally by the dashboard, by Claude DJ's tool calls, and by your scripts.

## The family

mixr is one of three small Rust tools by Chris McLennan:

- **mixr** — this DJ app
- **[tmnl](https://tmnl.sh)** — a GPU-rendered terminal that lets mixr render as a native tab (full graphics) instead of nested in some other terminal emulator
- **[mnml](https://mnml.sh)** — a NvChad-style terminal IDE

mixr runs fine in any terminal. Inside tmnl specifically, mixr gets native-tab integration via the `tmnl-protocol` cell-streaming layer.
